5. В видео подсказка про mac адреса
6. Имя устройства, которое хочет подключиться имеет вид: md5 MAC
7. Декодировать md5 и найти всех друзей Вини-пуха
8. Перевести MAC адрес друзей Вини-пуха в виде ascii
